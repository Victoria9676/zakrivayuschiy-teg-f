# Zakrivayuschiy-teg-f


## Description
A project that reflects memories along the way, which allows you to look at your condition and emotions at different stages of study.

In this project, we are first working on a static version, then on an adaptive one, and at the very end animation is enabled. This approach helps to perceive the project as a whole, rather than as a collection of individual parts.
